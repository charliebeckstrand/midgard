- Tailwind CSS v4 in apps/mimir: every workspace package whose source files contain Tailwind `className` strings must be explicitly listed with `@source '../../../packages/<name>/src'` in `globals.css`. This includes `rune`, `heimdall`, and any future package. Auto-detection only covers the app's own files. Missing a package silently omits those classes — rune worked first because it was added explicitly, while heimdall was missed.
