# Switch

A toggle for an immediate on/off setting — flip it and it takes effect at once.

## Overview

Under the hood a Switch is a real checkbox with a switch role, so it's keyboard- and screen-reader-native and works inside a form. It can be controlled from state or left to manage its own, and it integrates with the design system's form and field wrappers.

## Usage

Reach for a Switch when the change applies immediately rather than waiting behind a Save button — notification toggles, feature flags, visibility. Give it a label so its purpose is clear on its own; use a Checkbox instead when the choice is part of a set the user confirms together.
