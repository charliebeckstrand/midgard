# Switch

A toggle for an immediate on/off setting — flip it and it takes effect at once.

Reach for a Switch when the change applies immediately rather than waiting behind a Save button: notification toggles, feature flags, visibility. Under the hood it's a real checkbox with a switch role, so it's keyboard- and screen-reader-native and works inside a form, and it can be controlled from state or left to manage its own. Give it a label so its purpose is clear on its own.
