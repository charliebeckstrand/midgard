# Alert

A message bar that speaks up in place — feedback attached to the content it's about, not a transient toast.

Use an Alert to surface the outcome of an action, a warning that needs acknowledgement, or a standing notice. Its severity is semantic: the kind of message picks the color and icon and, more importantly, how assertively a screen reader announces it. It can carry a heading, a message, and its own controls, and be made dismissible when the notice is something the user should be able to clear. Reserve the assertive severities for messages worth interrupting someone over.
