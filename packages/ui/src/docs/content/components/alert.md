# Alert

A message bar that speaks up in place — feedback attached to the content it's about, not a transient toast.

## Overview

An Alert carries a semantic severity: the kind of message picks its color and icon and, more importantly, how assertively a screen reader announces it. It can hold a heading, a message, and its own actions, and it can be made dismissible when the notice is something the user should be able to clear.

## Usage

Use an Alert to surface the outcome of an action, a warning that needs acknowledgement, or a standing notice. Reserve the assertive severities for messages worth interrupting someone over; use the quieter ones for confirmations and passing information.
